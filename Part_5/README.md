# Part 5｜最佳化與控制（Optimization / BO / RL）

本 Part 的目標是：把「最佳化」從工程模型出發，延伸到「智慧實驗設計（貝葉斯最佳化）」與「強化學習控制」，形成一條完整的決策主線。

- `Part_5/Unit04_Reaction_Kinetics.md` / `Part_5/Unit04_Reaction_Kinetics.ipynb`：反應動力學模擬與最佳反應時間（以模擬帶出最佳化直覺）
- `Part_5/Unit05_Process_Optimization.md` / `Part_5/Unit05_Process_Optimization.ipynb`：數值最佳化（SciPy minimize，利潤地形與靈敏度）
- `Part_5/Unit10_Bayesian_Optimization.md` / `Part_5/Unit10_Bayesian_Optimization.ipynb`：貝葉斯最佳化（智慧實驗設計：代理模型 + acquisition）
- `Part_5/Unit10_README.md`：Unit10 使用指南（講師備註，可選讀）
- `Part_5/Unit10_Enhancement_Summary.md`：Unit10 增補總結（講師備註，可選讀）
- `Part_5/Unit14_RL_Control.md` / `Part_5/Unit14_RL_Control.ipynb`：強化學習控制（Q-Learning / SARSA，離散控制）
- `Part_5/Unit14_RL_Control_SAC.ipynb`：深度強化學習控制（SAC，連續控制；承接 Unit14 章節 5.2）
