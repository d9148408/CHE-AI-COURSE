# Unit02 Python程式語言基礎 (Python Programming Basics)

## 📚 單元簡介

Python 是目前資料科學、機器學習與人工智慧領域最受歡迎的程式語言。其語法簡潔易讀、生態系統豐富，且擁有強大的科學計算與機器學習套件支援，是學習AI技術的最佳入門語言。

本單元將帶您從零開始學習 Python 程式語言的核心概念，包括基本語法、資料型態、流程控制、函式定義、以及例外處理等重要主題。透過化工相關的實例（如溫度轉換、反應計算、資料處理等），您將學會如何使用 Python 解決實際問題。

完成本單元後，您將具備紮實的 Python 程式設計能力，為後續的資料處理、機器學習課程打下堅實基礎。本單元適合完全沒有程式設計經驗的學員，也歡迎有其他語言基礎的學員快速複習 Python 特性。

## 📝 Python重要概念總結

### 語法特性
- **縮排**：使用4個空格定義程式碼區塊（不是選項，是語法要求）
- **動態型態**：變數不需宣告型態，自動推斷
- **物件導向**：所有資料都是物件

### 資料結構選擇
- **List**：需要有序、可變動的集合 → `[1, 2, 3, 4]`
- **Tuple**：需要不可變的序列 → `(1, 2, 3, 4)`
- **Dictionary**：需要鍵值對映射 → `{'temp': 25, 'pressure': 101}`
- **Set**：需要唯一元素集合 → `{1, 2, 3, 4}`

### 迴圈選擇
- **for迴圈**：已知迭代次數或遍歷序列 → `for i in range(10):`
- **while迴圈**：條件未知，需持續執行直到條件不成立 → `while temp < 100:`

### 函式設計原則
- **單一職責**：一個函式只做一件事
- **有意義的命名**：函式名稱應清楚表達功能
- **適當的參數**：不要過多參數（建議≤5個）
- **返回值**：明確返回計算結果

## 🎓 Python vs 其他語言

### Python vs MATLAB（化工常用）
| 特性 | Python | MATLAB |
|------|--------|--------|
| **成本** | 免費開源 | 商業授權 |
| **語法** | 簡潔直觀 | 專注於矩陣運算 |
| **生態系統** | 通用性強，AI/ML套件豐富 | 工程計算專精 |
| **學習曲線** | 平緩 | 中等 |
| **工業應用** | 廣泛（Google、Netflix） | 學術與工程界 |

### Python vs C/C++
| 特性 | Python | C/C++ |
|------|--------|-------|
| **執行速度** | 較慢（解釋型語言） | 非常快（編譯型語言） |
| **開發速度** | 快速 | 較慢 |
| **記憶體管理** | 自動 | 手動 |
| **適用場景** | 原型開發、資料分析、AI | 系統程式、高效能計算 |

**結論**：Python在AI/ML領域的優勢在於快速開發、豐富的套件生態，以及易於學習與維護。
