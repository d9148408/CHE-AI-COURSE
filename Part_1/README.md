# Part 1: Python基礎與資料處理 (Python Basics and Data Processing)

## 📚 Part 1 簡介

Part 1 是「AI在化工上的應用」課程的基礎部分，專為化學工程學系的學生設計。本部分將從AI與機器學習的基本概念開始，逐步建立Python程式設計能力、科學計算工具的應用，以及資料視覺化技能。這些基礎知識與技能是後續學習機器學習、深度學習等進階主題的必要條件。

本部分共包含四個單元，涵蓋從AI理論基礎、Python語言核心、數值與表格資料處理、到資料視覺化的完整學習路徑。透過化工領域的實際案例與練習，您將學會如何運用這些工具解決化工資料分析的實際問題。

## 💡 學習建議

1. **循序漸進**：按照單元順序學習，每個單元都是後續學習的基礎
2. **實作導向**：在學習理論的同時，務必動手執行範例程式碼
3. **完成作業**：透過作業練習鞏固所學，發現知識盲點
4. **化工應用**：思考如何將所學工具應用於自己的化工領域研究或工作
5. **建立筆記**：記錄重要概念與常用語法，方便日後查閱
