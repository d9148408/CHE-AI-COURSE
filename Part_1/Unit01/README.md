# Unit01 AI與機器學習概論 (AI and Machine Learning Overview)

## 📚 單元簡介

本單元是「AI在化工上的應用」課程的開端，專為化學工程學系的學生設計。在數位轉型與智慧製造的浪潮下，人工智慧與機器學習已成為化工產業提升競爭力的關鍵技術。本單元將帶您認識AI與機器學習的基礎概念、發展歷程，以及在化工領域的應用實例。

本單元涵蓋從AI的歷史演進、機器學習的基本分類（監督式、非監督式、強化式學習），到完整的機器學習工作流程。我們將介紹課程的整體架構，幫助您了解從Python基礎、資料處理、機器學習演算法，到深度學習與進階應用的完整學習路徑。透過本單元，您將建立對AI技術的全面認識，並了解如何將這些技術應用於化工領域的實際問題。

本單元為純理論課程，著重於建立完整的知識框架與學習地圖，為後續的實作單元奠定基礎。

---

## 🎯 學習目標

完成本單元後，您將能夠：

1. **理解AI與機器學習的核心概念**：掌握人工智慧、機器學習、深度學習的定義與關係
2. **認識機器學習的類型與應用**：理解監督式、非監督式、強化式學習的差異與適用場景
3. **掌握機器學習工作流程**：了解從問題定義、資料處理、模型訓練到部署的完整流程
4. **認識化工領域的AI應用**：理解AI技術在製程優化、品質預測、異常檢測等化工場景的應用潛力
5. **建立課程學習路徑**：清楚了解本課程的架構安排與學習目標

## 🔍 化工領域核心應用

### 1. 製程優化與控制 ⭐
- **目標**：透過AI模型找出最佳操作條件，提升產率、降低成本
- **方法建議**：監督式學習（回歸模型）、強化式學習
- **關鍵技術**：
  - 製程數據收集與前處理
  - 多變數優化
  - 即時控制策略

### 2. 品質預測與管理
- **目標**：根據製程參數預測產品品質，提早調整避免不良品
- **方法建議**：監督式學習（回歸、分類）
- **關鍵技術**：
  - 特徵工程與變數選擇
  - 軟測量（Soft Sensor）
  - 線上品質預測

### 3. 異常檢測與預測性維護
- **目標**：及早發現設備異常，預防停機事故
- **方法建議**：非監督式學習（異常檢測）、監督式學習（故障分類）
- **關鍵技術**：
  - 時間序列分析
  - 異常模式識別
  - 剩餘使用壽命預測

### 4. 配方設計與溶劑篩選
- **目標**：加速新產品開發，減少實驗成本
- **方法建議**：監督式學習、貝氏優化
- **關鍵技術**：
  - 化學資訊學（Cheminformatics）
  - 分子性質預測
  - 實驗設計優化

### 5. 反應條件優化
- **目標**：找出最佳反應溫度、壓力、催化劑配比等條件
- **方法建議**：貝氏優化、強化式學習
- **關鍵技術**：
  - 反應動力學建模
  - 多目標優化
  - 實驗次數最小化
